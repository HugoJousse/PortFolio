Veille Technologique

Definition

La veille technologique est un procédé qui consiste à s'informer de façon sistématique sur les technologies les plus récentes sur un sujet précis

Mon sujet de Veille

Ma veille technologique est concentré sur les casques de réalité virtuelle (casque VR).
Mon objectif est de trouver, à partir d'un problème constaté lors du développement d'un casque VR, la technologie qui a été intégré pour résoudre ce problème. Ma veille c'est beaucoup axé sur le plan physique du casque. (L'écran, les lentilles, capteurs...)

Pourquoi avoir choisi cette veille ?

Les casques VR sont intéressant. Ils sont maitenant utilisé dans divers domaines, autant dans le jeu vidéo que dans le domaine médicale. Les développements d'applications ou jeux en lien avec un casque VR sont de plus en plus nombreux. Dans mon domaine, il est donc intéressant de se pencher sur ces technologies.

Methodes de recherche

Flux RSS

Divers flux RSS comme Google Alerts peuvent etre utiliser durant une veille technologique. Je me suis concentré sur les Googles Alerts en retrants divers mots clés comme "Casques VR", "Fonctionnement Casque VR", "Avenir de la VR"
