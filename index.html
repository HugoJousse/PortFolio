Veille Technologique

Définition

La veille technologique est un procédé qui consiste à s'informer de façon systématique sur les technologies les plus récentes sur un sujet précis. Ainsi, il faut définir un cadre de recherche, puis trier les informations pour enfin stocker les données pertinentes.

Mon sujet de veille

Ma veille technologique est concentrée sur les casques de réalité virtuelle (casque VR).
Mon objectif est de trouver, à partir d'un problème constaté lors du développement d'un casque VR, la technologie qui a été intégrée pour résoudre ce problème. Ma veille s'est beaucoup axée sur le plan physique du casque. (L'écran, les lentilles, capteurs...)

Pourquoi avoir choisi cette veille ?

Les casques VR sont intéressants. Ils sont maitenant utilisés dans divers domaines, autant dans le jeu vidéo que dans le domaine médicale. Les développements d'applications ou jeux en lien avec un casque VR sont de plus en plus nombreux. Dans mon domaine, il est donc intéressant de se pencher sur ces technologies.

Méthodes de recherches

1 - Flux RSS

Divers flux RSS comme Google Alerts peuvent être utilisés durant une veille technologique. Je me suis concentré sur les Google Alerts en rentrants divers mots clés comme "Casques VR", "Fonctionnement Casque VR", "Avenir de la VR".



2 - Recherches classiques



L'utilisation des moteurs de recherche a aussi été utile pour cette veille. J'ai pu affiner mon travail de veille.

Intérêt d'une veille

Une veille permet de rester à jour des dernières technologies dans un secteur donné. Dans une entreprise cela peut permettre de prendre de l'avance sur des concurrents ainsi que de rester toujours performant.

Patrimoine informatique

Le patrimoine informatique regroupe l'ensemble des ressources matérielles et imatérielles informatiques d'un domaine comme une entreprise.

Importance de la gestion du patrimoine informatique

Il faut pouvoir gérer les composants d'un parc informatique tel que les appareils, serveurs, logiciels (licences, antivirus...). De plus, il est nécessaire de garder une trace de chaques actions importantes effectuées sur un parc informatique. Les procédures, modifications et autres données doivent être archivées. Ce sont des ressources vitales appartenant au patrimoine informatique.

Comment gérer un patrimoine informatique ?

Divers logiciels de gestion de parc informatiques existent sur le marché. Ils ont plusieurs rôles :
- Gérer le suivi et l'affectation du matériel informatique
- Gérer les équipements de prêt
- Gérer les licences
- Assister les utilisateurs
- Etablir une base de connaissance
